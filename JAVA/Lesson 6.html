Lesson 6: Arrays

Overview:

An array in Java is a collection of variables of the same type stored in contiguous memory locations. It allows efficient storage and retrieval of data, making it a fundamental data structure in Java programming.

Key Characteristics of Arrays

  1. Fixed Size: The size of an array is specified at the time of its creation and cannot be changed.
  2. Index-Based: Elements in an array are accessed using an index, starting from 0.
  3. Homogeneous Data: Arrays store elements of the same data type.
  4. Random Access: Any element can be accessed directly using its index.

Code Example:

public class HelloWorld // {
public static void main(String[] args)
{

// Single-dimensional array
int[] numbers = {10, 20, 30, 40, 50};
System.out.println("Accessing Elements:");
System.out.println("First Element: " + numbers[0]);
System.out.println("Last Element: " + numbers[numbers.length - 1]);
System.out.println("\nIterating using for loop:");
for (int i = 0; i < numbers.length; i++) {
System.out.println("Element at index " + i + ": " + numbers[i]); }
System.out.println("\nIterating using enhanced for loop:");
for (int num : numbers) {
System.out.println(num); }
// Multi-dimensional array
int[][] matrix = {
{1, 2, 3},
{4, 5, 6},
{7, 8, 9} }; System.out.println("\nAccessing Multi-Dimensional Array:"); System.out.println("Element at [1][2]: " + matrix[1][2]); } }
