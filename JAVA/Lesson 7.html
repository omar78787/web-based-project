Lesson 7: Object-Oriented Programming (OOP)

Overview:

Object-Oriented Programming (OOP) is a paradigm centered around objects and classes. It emphasizes the creation and manipulation of objects that represent real-world entities, enabling modular, reusable, and scalable code.

Core Principles of OOP

  1. Encapsulation: Encapsulation hides the internal state of an object and restricts access to it through public methods. This ensures controlled access and enhances security.
  2. Inheritance: Inheritance allows a class (child) to inherit fields and methods from another class (parent). This promotes code reusability.
  3. Polymorphism: Polymorphism allows objects to take multiple forms, enabling flexibility and dynamic method invocation.
  4. Abstraction: Abstraction hides implementation details and shows only the essential features of an object. This is achieved using abstract classes or interfaces.

Code Example:

public class HelloWorld // {
public static void main(String[] args)
{
// Encapsulation
Student student = new Student();
student.setName("John");
System.out.println("Student Name: " + student.getName());
// Inheritance
Dog dog = new Dog();
dog.eat();
dog.bark();
// Polymorphism
Animal animal = new Cat();
animal.sound();
// Abstraction
Shape shape = new Circle();
shape.draw();
// Interface
Vehicle vehicle = new Car();
vehicle.move(); } }

Output:


Student Name: John
This animal eats food.
The dog barks.
Meow
Drawing a circle.
The car moves.
