Lesson 5: Loops

Overview:

Loops in Java allow code to be executed repeatedly based on a condition. They are essential for tasks like iterating through arrays, processing input, or automating repetitive operations.

Types of Loops in Java

    for Loop: The for loop is used when the number of iterations is known beforehand.
  1. while Loop: The while loop checks a condition before executing the block of code. It is used when the number of iterations is not known beforehand.
  2. do-while Loop: The do-while loop executes the code block once before checking the condition. It ensures the code runs at least once.

Code Example:


public class LoopsExample { public static void main(String[] args) { // Using a for loop
for (int i = 1; i <= 5; i++) {
System.out.println("For Loop: " + i);
}
// Using a while loop
int count = 1;
while (count <= 3) {
System.out.println("While Loop: " + count);
count++;
}
// Using a do-while loop
int num = 1;
do {
System.out.println("Do-While Loop: " + num);
num++;
} while (num <= 2);
// Using a break statement
for (int i = 1; i <= 5; i++) {
if (i == 4) {
break; }
System.out.println("Break Example: " + i); }
// Using a continue statement
for (int i = 1; i <= 5; i++) {
if (i == 3) {
continue;
}
System.out.println("Continue Example: " + i);
} } }
