Lesson 8: Strings

Overview:

Strings in Java represent sequences of characters. They are widely used for text manipulation and are objects of the String class, part of the java.lang package.

Characteristics of Strings in Java

  1. Immutable: Once a String object is created, it cannot be changed.
  2. Stored in the String Pool: Java optimizes memory usage by storing strings in a common pool.
  3. Built-in Methods: Java provides a variety of methods for string manipulation.

Code Example:

public class HelloWorld // {
public static void main(String[] args)
{

String message = "Hello, Java!";
System.out.println("Original String: " + message);
// Length
System.out.println("Length: " + message.length());
// Concatenation
System.out.println("Concatenated String: " + message.concat(" Welcome!"));
// Substring
System.out.println("Substring: " + message.substring(7, 11));
// Character at index
System.out.println("Character at Index 1: " + message.charAt(1));
// Case conversion
System.out.println("Uppercase: " + message.toUpperCase());
System.out.println("Lowercase: " + message.toLowerCase());
// Replacing characters
System.out.println("Replaced String: " + message.replace("Java", "World"));
// Using StringBuilder
StringBuilder sb = new StringBuilder("Java");
sb.append(" Programming");
System.out.println("StringBuilder: " + sb.toString()); } }

Output:


Original String: Hello, Java!
Length: 12
Concatenated String: Hello, Java! Welcome!
Substring: Java
Character at Index 1: e
Uppercase: HELLO, JAVA!
Lowercase: hello, java!
Replaced String: Hello, World!
StringBuilder: Java Programming
