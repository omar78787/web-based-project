Lesson 3: Operators in Java

Overview

Operators in Java are symbols or keywords used to perform operations on variables and values. Java offers a wide range of operators to perform mathematical computations, comparisons, and logical decisions.

Types of Operators in Java

  1. Arithmetic Operators: Perform basic mathematical operations. (+-*/%)
  2. Relational (Comparison) Operators: Compare two values and return a boolean result.(== != > < >= <=)
  3. Logical Operators: Used to combine multiple boolean expressions.( && `!)
  4. Assignment Operators: Assign values to variables.(+ - ++ --)
  5. Unary Operators: Work with a single operand.

Code Example:

public class HelloWorld // {
public static void main(String[] args)
{
int a = 10, b = 5;
boolean condition = (a > b);
// Arithmetic operations
System.out.println("Sum: " + (a + b)); // 15
System.out.println("Difference: " + (a - b)); // 5
// Relational and Logical
System.out.println("Is a > b? " + condition); // true
System.out.println("Is a < b? " + (a < b)); // false
System.out.println("Logical AND: " + (condition && (b > 0))); // true
// Assignment
a += 2; // a = a + 2
System.out.println("New a: " + a); // 12 } }

Output:

Sum: 15
Difference: 5
Is a > b? true
Is a < b? false
Logical AND: true
New a: 12
