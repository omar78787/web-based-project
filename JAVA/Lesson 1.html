Lesson 1 Introduction To Java

Overview

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). Itâ€™s widely used because of its platform independence, security, and versatility. Java programs are compiled into bytecode, which can run on any Java Virtual Machine (JVM), making Java "Write Once, Run Anywhere" (WORA).

Why Learn JAVA?

  1. Versatility: Used in web development, mobile applications (Android), backend systems, and desktop applications.
  2. Popularity: Ranked among the top programming languages worldwide.
  3. Robust Libraries: Extensive APIs and libraries for almost every need.
  4. Job Opportunities: A highly sought-after skill in the tech industry.

First Code Example In JAVA:

public class HelloWorld // {
public static void main(String[] args)
{ // Print message to the console System.out.println("Hello, World!"); // This'll output Hello, World! to the console}}

Output:

Hello, World!

Code Explanation:

  1. public class HelloWorld:
    • Declares a class named HelloWorld. The class name must match the file name.
  2. public static void main(String[] args):
    • The main method is the entry point for Java programs. It must always have this exact signature.
  3. System.out.println("Hello, World!");:
    • Outputs text to the console.
