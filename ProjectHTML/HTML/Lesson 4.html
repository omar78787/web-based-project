Lesson 4: HTML Text Formatting

Overview

Text formatting in HTML allows you to style and structure the content on a webpage. HTML provides various tags to apply styles like bold, italics, and underline, as well as organizing text into paragraphs, headings, and lists.

First Code Example In HTML:

<html>
<body>
<h1>HTML Text Formatting Example</h1>
<p>This is a paragraph with <b>bold</b>, <i>italic</i>, and <u>underlined</u> text.</p>
<p align="center">This paragraph is centered.</p>
<br>
<p>Here are some lists:</p>
<ul>
<li>First unordered item</li>
<li>Second unordered item</li>
</ul>
<ol>
<li>First ordered item</li>
<li>Second ordered item</li>
</ol>
</body>
</html>

Output:

HTML Text Formatting Example

This is a paragraph with bold, italic, and underlined text.

This paragraph is centered.


Here are some lists:

  • First unordered item
  • Second unordered item
  1. First ordered item
  2. Second ordered item
