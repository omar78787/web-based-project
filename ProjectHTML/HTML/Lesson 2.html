Lesson 2: HTML Elements and Attributes

Overview

In HTML, content is structured using elements. These elements are defined by tags, and each element can have attributes that provide additional information about the element. Understanding how elements and attributes work together is fundamental to creating well-structured and functional web pages.

First Code Example In HTML:


<!-- Example of HTML Elements -->
<!-- An element typically consists of an opening tag, content, and a closing tag. -->
<h1>Welcome to HTML!</h1>
<p>This is a paragraph inside the body of the web page.</p>
<!-- Example of Self-Closing Elements -->
<!-- Some elements like <img> donâ€™t require a closing tag. -->
<img src="image.jpg" alt="A beautiful image">
<br>This line breaks here because of the <br> tag.
<!-- Example of HTML Attributes -->
<!-- Attributes provide additional information about an element. -->
<a href="https://www.example.com">Visit Example</a>
<!-- Example of Boolean Attributes -->
<!-- Boolean attributes like <input>'s checked attribute are either present or not. -->
<input type="checkbox" checked> I agree to the terms and conditions
<!-- Example of Grouping Elements -->
<!-- Elements can be grouped using tags like <div> and <span> -->
<div>
This is inside a div element.
</div>

Explanation:

  • HTML Elements: Elements like <h1>, <p>, and others are shown with an opening tag, content, and closing tag.
  • Self-Closing Elements: The <img> and <br> tags do not require closing tags.
  • Attributes: The <a> tag uses the href attribute, and the <input> tag uses the checked attribute.
  • Boolean Attributes: The checked attribute in the checkbox input is either present or absent
  • Grouping Elements: The <div> element groups the paragraph inside it.
