Lesson 10: Object-Oriented Programming (OOP) in C++

Overview

Object-Oriented Programming (OOP) is a programming paradigm that is centered around objects and classes. C++ is one of the most widely used OOP languages, enabling developers to model real-world problems using entities (objects) that have attributes (data members) and behaviors (functions or methods). OOP enhances modularity, code reuse, and maintainability.

Core Concepts of OOP

  • Classes and Objects
    1. Class: A blueprint or prototype for creating objects (instances).
    2. Object: An instance of a class. It is created using the class as a template.

Code Example:

class Car {
public:
// Attributes (data members)
std::string brand;
int year;
// Methods (functions)
void drive() {
std::cout << "The " << brand << " car is driving." << std::endl;
} };
int main() {
// Creating an object of class Car
Car myCar;
myCar.brand = "Toyota";
myCar.year = 2020;
myCar.drive(); // Call method on the object
return 0; }

Output:

The Toyota car is driving.
