Lesson 3: Operators in C++

Overview

Operators in C++ are symbols used to perform operations on variables and values. They are fundamental for arithmetic computations, comparisons, and logical decisions.

Types of Operators in C++

  1. Arithmetic Operators Perform basic mathematical operations.(+-*/%)
  2. Relational (Comparison) Operators Compare two values and return a boolean result.(== !=>< >=<=)
  3. Logical Operators Combine multiple boolean expressions. ( && ` !)
  4. Assignment Operators Assign values to variables. (= += -=)
  5. Unary Operators Operate on a single operand (+ - ++ --)
  6. Bitwise Operators Perform operations at the bit level (& ` ^ ~)

Operator Precedence

  1. Parentheses ()
  2. Unary operators +, -, ++, --
  3. Arithmetic operators *, /, %
  4. Relational operators <, >, ==
  5. Logical operators &&, ||
  6. Assignment =, +=, -=

Code Example:

#include
int main() {
int a = 10, b = 5;
// Arithmetic operations
std::cout << "Sum: " << (a + b) << std::endl;
std::cout << "Difference: " << (a - b) << std::endl;
// Relational and Logical
std::cout << "Is a > b? " << (a > b) << std::endl;
std::cout << "Logical AND: " << ((a > b) && (b > 0)) << std::endl;
// Assignment
a += 2; // a = a + 2
std::cout << "New a: " << a << std::endl;
// Bitwise
std::cout << "Bitwise AND: " << (a & b) << std::endl;
return 0; }

Output:

Sum: 15
Difference: 5
Is a > b? 1
Logical AND: 1
New a: 12
Bitwise AND: 4
