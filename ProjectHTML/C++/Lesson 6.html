Lesson 6: Functions in C++

Overview

Functions in C++ are reusable blocks of code designed to perform a specific task. They allow for better modularity, code readability, and reusability in programs.

Components of a Function

  1. Function Declaration (Prototype): Declares the function's name, return type, and parameters without defining its body.
  2. Function Definition: Contains the actual code or logic of the function.
  3. Function Call: Executes the function by passing the required arguments.

Code Example:

#include
int add(int a, int b) {
return a + b; }
int main() {
int result = add(5, 3);
std::cout << "Sum: " << result << std::endl;
return 0; }

Output:

Sum: 8
