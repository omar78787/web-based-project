Lesson 2: Variables and Data Types

Overview

Variables in Java are containers for storing data values. They are declared with a specific data type, determining the kind of data the variable can hold. Data types in Java are categorized into primitive types and reference types.

Primitive Data Types

  1. byte: 8-bit integer, range: -128 to 127
  2. short: 16-bit integer, range: -32,768 to 32,76
  3. int: 32-bit integer, range: -2^31 to 2^31 - 1.
  4. long: 64-bit integer, range: -2^63 to 2^63 - 1
  5. float: 32-bit floating-point number
  6. double: 64-bit floating-point number (more precise than float)
  7. char: A single 16-bit Unicode character
  8. boolean: Represents true or false.

Code Example:

public class HelloWorld // {
public static void main(String[] args)
{ // Primitive types
int age = 30;
double salary = 45000.75;
char initial = 'J';
boolean isEmployed = true;
// Reference types
String name = "Alice";
// Printing values
System.out.println("Name: " + name);
System.out.println("Age: " + age);
System.out.println("Salary: $" + salary);
System.out.println("Initial: " + initial);
System.out.println("Employed: " + isEmployed);
} }

Output:

Name: Alice
Age: 30
Salary: $45000.75
Initial: J
Employed: true

Rules:

Must start with a letter, $, or _, Cannot use reserved keywords like class or int.
