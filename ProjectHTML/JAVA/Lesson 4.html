Lesson 4: Conditional Statements

Overview:

Conditional statements allow a program to make decisions based on specific conditions. These statements enable the flow of execution to branch into different paths depending on whether a condition evaluates to true or false.

Types of Conditional Statements in Java

  1. if Statement The if statement evaluates a condition and executes a block of code only if the condition is true.
  2. if-else Statement The if-else statement provides an alternate block of code to execute when the condition is false.
  3. if-else if-else Ladder Use multiple conditions to evaluate and execute specific blocks of code.

Code Example:

public class HelloWorld // {
public static void main(String[] args)
{
int temperature = 30;
// Using if-else
if (temperature > 35) {
System.out.println("It's very hot outside.");
} else if (temperature > 25) {
System.out.println("It's warm outside."); }
else {
System.out.println("It's cold outside."); } } }

Output:

It's warm outside.
