Lesson 7: Arrays in C++

Overview

An array in C++ is a collection of elements of the same data type stored in contiguous memory locations. Arrays are useful for managing large amounts of data efficiently under a single variable name.

Types of Arrays

  1. Single-Dimensional Arrays
  2. Multi-Dimensional Arrays
  3. Dynamic Arrays

Code Example:

#include #include
int main() {
int arr[5] = {50, 20, 40, 10, 30};
// Display original array
std::cout << "Original array: ";
for (int i = 0; i < 5; i++) {
std::cout << arr[i] << " "; }
std::cout << std::endl;
// Sort the array
std::sort(arr, arr + 5);
// Display sorted array
std::cout << "Sorted array: ";
for (int i = 0; i < 5; i++) {
std::cout << arr[i] << " "; }
std::cout << std::endl;
// Search for an element
int key = 20;
int* result = std::find(arr, arr + 5, key);
if (result != arr + 5) {
std::cout << "Element " << key << " found at index " << (result - arr) << std::endl;
} else {
std::cout << "Element " << key << " not found." << std::endl; }
return 0; }

Output:

Original array: 50 20 40 10 30
Sorted array: 10 20 30 40 50
Element 20 found at index 1
