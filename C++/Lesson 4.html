Lesson 4: Conditional Statements in C++

Overview

Conditional statements allow a program to make decisions based on conditions. These conditions are evaluated as true or false, and the program executes specific code blocks accordingly.

Types of Conditional Statements in C++

  1. if Statement
  2. if-else Statement
  3. if-else if-else Ladder
  4. Nested if Statements

Code Example:

#include
int main() {
int num;
std::cout << "Enter a number: ";
std::cin >> num;
if (num > 0) {
std::cout << "The number is positive." << std::endl;
} else if (num < 0) {
std::cout << "The number is negative." << std::endl;
} else {
std::cout << "The number is zero." << std::endl; }
return 0; }

Output:

Enter a number: 5
The number is positive.
