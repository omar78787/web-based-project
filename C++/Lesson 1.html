Lesson 1: Introduction to C++

Overview

C++ is a powerful, high-performance programming language that combines object-oriented, procedural, and generic programming features. Originally developed by Bjarne Stroustrup in 1983 as an enhancement to C, C++ has become widely used for system/software development, game development, and real-time systems.

Key Features of C++

  1. Object-Oriented: Supports classes, inheritance, polymorphism, encapsulation, and abstraction.
  2. Low-Level Manipulation: Retains the low-level control of C with added features.
  3. Rich Library Support: Extensive libraries for various purposes, including the Standard Template Library (STL).
  4. Portability: Programs written in C++ are platform-independent, requiring only recompilation on a new platform.
  5. Performance: Optimized for high-performance applications.

Why Learn C++?

  1. It is foundational for understanding other programming languages.
  2. C++ is used in industries like game development, financial modeling, and operating system development.
  3. It balances the speed and efficiency of C with advanced features like templates and exception handling.

First Code Example In C++:

#include int main() {
// Declare variables
int age = 30;
double salary = 75000.50;
char grade = 'A';
bool isEmployed = true;
// Display values
std::cout << "Age: " << age << std::endl;
std::cout << "Salary: $" << salary << std::endl;
std::cout << "Grade: " << grade << std::endl;
std::cout << "Employed: " << (isEmployed ? "Yes" : "No") << std::endl; return 0; }

Output:

Age: 30 Salary: $75000.5 Grade: A Employed: Yes

Notes:

  1. Every C++ program starts with a main function.
  2. Use std::cout and std::cin for basic input/output.
  3. Use comments using // to make your code easier to understand.
  4. There is a way to make it so you don't need to type std:: but it's bad practice
