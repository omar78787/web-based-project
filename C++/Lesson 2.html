Lesson 2: Variables and Data Types

Overview

Variables in C++ are used to store data that can be manipulated throughout the program. Each variable has a specific type that determines the kind of data it can hold.

Primitive Data Types:

  • int: Stores integers. Example: int age = 25;
  • float: Stores floating-point numbers (single precision). Example: float weight = 65.5;
  • double: Stores floating-point numbers (double precision). Example: double distance = 123.456;
  • char: Stores a single character. Example: char grade = 'A';
  • bool: Stores true or false. Example: bool isPassed = true;

Input in C++ is done by cin and output is done by cout, cin can take in multiple variables.

Basic input and output Code Example:

#include int main() { #include
int main() {
int a, b;
std::cout << "Enter two numbers: ";
std::cin >> a >> b;
std::cout << "Sum: " << (a + b) << std::endl;
return 0; } }
