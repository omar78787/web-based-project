Lesson 5: Loops in C++

Overview

Loops in C++ allow you to execute a block of code repeatedly based on a condition. They are a fundamental construct for automating repetitive tasks, iterating through data, or performing calculations until a certain condition is met.

Types of Loops in C++

  1. for loop
  2. while loop
  3. do-while loop

Code Example:

#include
int main() {
// For loop example
std::cout << "For Loop:" << std::endl;
for (int i = 1; i <= 3; i++) {
std::cout << i << " "; }
std::cout << std::endl;
// While loop example
std::cout << "While Loop:" << std::endl;
int count = 1;
while (count <= 3) {
std::cout << count << " ";
count++; }
std::cout << std::endl;
// Do-while loop example
std::cout << "Do-While Loop:" << std::endl;
int num = 1;
do {
std::cout << num << " ";
num++;
}
while (num <= 3);
std::cout << std::endl;
return 0; }

Output:


For Loop:
1 2 3
While Loop:
1 2 3
Do-While Loop:
1 2 3
