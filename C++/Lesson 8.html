Lesson 8: Pointers in C++

Overview

Pointers are variables that store the memory address of another variable. They are a powerful feature of C++ that allow direct manipulation of memory, facilitating dynamic memory allocation, efficient data structures, and performance optimizations.

Features of Pointers

  1. Features of Pointers
  2. Pointers can point to any data type (e.g., int, char, float).
  3. They enable dynamic memory allocation.
  4. Pointers are used for passing large structures to functions efficiently.

Code Example:

#include
void swap(int* a, int* b) {
int temp = *a;
*a = *b;
*b = temp; }
int main() {
int x = 10, y = 20;
std::cout << "Before swap: x = " << x << ", y = " << y << std::endl;
swap(&x, &y);
std::cout << "After swap: x = " << x << ", y = " << y << std::endl;
return 0; }

Output:

Before swap: x = 10, y = 20 After swap: x = 20, y = 10
