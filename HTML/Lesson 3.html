Lesson 3: HTML Document Structure

Overview

An HTML document consists of specific elements that define its structure. These elements ensure the proper organization of the content for browsers and search engines. Let's look at the key components.

Basic HTML Document Structure

  1. The head Section

    2. The head section provides meta-information about the HTML document. It is not displayed on the page itself, but it contains important elements for the browser.

    • <meta charset="UTF-8">: Specifies the character encoding for the document.
    • <title>: The title appears in the browser tab and search engine results.
    • <link>: Links external resources like stylesheets
  2. The body Section

    3. The body section holds the content that is visible on the web page. This is where you place elements like headings, paragraphs, images, links, and more.

Example Document with head and body

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>My First Web Page</title>
</head>
<body>
<h1>Welcome to My Web Page</h1>
<p>This is an example of a simple HTML document structure.</p>
</body>
</html>

Try this on your code editor (you can even use it on notepad).
