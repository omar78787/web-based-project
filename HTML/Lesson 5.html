Lesson 5: HTML Links and Images

Overview

Links and images are essential for making web pages interactive and visually appealing. In this lesson, we will cover how to create hyperlinks and add images to a webpage.

Links in HTML

Hyperlinks are created using the <a> (anchor) tag. Links are used to navigate between web pages, websites, or to different sections within the same page.

Images in HTML

Images are added using the <img> tag. The src attribute specifies the image source (URL or file path), and the alt attribute provides alternative text for accessibility purposes.

Example In HTML:

<html>
<body>
<h1>HTML Links and Images Example</h1>
<a href="https://www.asu.edu.jo/en/Pages/default.aspx">Visit Example</a>
<p>Here is an image:</p>
<img src="../img/ISO_C++_Logo.svg.png" alt="HTML" width="250" height="250">
<p><a href="../java.html"> < img src="../img/java-logo.png" width="400"></a></p>
</body>
</html>
